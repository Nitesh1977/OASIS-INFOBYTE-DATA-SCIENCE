# Task 01 - Exploratory Data Analysis (EDA)

## Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on a dataset to understand patterns, relationships, and important insights using Python.

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Steps
1. Imported required libraries
2. Loaded the dataset
3. Performed basic data understanding
4. Checked missing and duplicate values
5. Generated statistical summary
6. Created visualizations and graphs
7. Analyzed relationships between variables
8. Derived meaningful insights

## Visualizations Included
- Correlation Heatmap
- Distribution Plot
- Bar Graph
- Scatter Plot

## Key Insights
- Higher fat items generally contain more calories.
- Some categories have significantly higher calorie values.
- Correlation heatmap helped identify relationships between variables.

## Conclusion
This project helped in understanding the importance of Exploratory Data Analysis in identifying trends, cleaning data, and generating business insights from raw datasets.
