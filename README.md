Performed Exploratory Data Analysis on a dataset using visualization techniques and statistical analysis to identify patterns and insights.
### Key Concepts:
- Data Cleaning
- Correlation Heatmap
- Distribution Analysis
- Scatter Plot
- Data Visualization

## Task 04 - Data Cleaning
Cleaned and preprocessed raw Twitter text data by handling missing values, duplicate records, and text preprocessing.
### Key Concepts:
- Missing Value Handling
- Duplicate Removal
- Text Cleaning
- Data Preprocessing

## Task 06 - Wine Quality Prediction
Built a Machine Learning model to predict wine quality based on various chemical properties.
### Key Concepts:
- Classification
- Random Forest Classifier
- Model Training
- Prediction
- Accuracy Evaluation

## Task 07 - Fraud Detection
Developed a fraud detection model to identify fraudulent transactions using Machine Learning techniques.
### Key Concepts:
- Classification
- Fraud Analysis
- Random Forest
- Model Evaluation
- Data Analysis

## Task 08 - Google Play Store App Analysis
Analyzed Google Play Store apps dataset to understand app categories, installs, ratings, and market trends.
### Key Concepts:
- Data Cleaning
- Category Analysis
- Ratings Analysis
- Install Trends
- Data Visualization
